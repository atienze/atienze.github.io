Sierpinski Triangle

Languages: Python

November, 2022 - December, 2022

Brief Description

This project involved implementing a graphical representation of the Sierpinski triangle using the Chaos Game algorithm. The code utilizes Python's turtle library to dynamically draw the fractal pattern by iterating through random point selections and midpoint calculations. The Sierpinski triangle is generated by selecting random corners of the triangle and repeatedly calculating midpoints to create a fractal shape. A custom color gradient was applied to the points based on their proximity to the corners, enhancing the visual appeal of the generated fractal. The final output was generated through a loop that updated the canvas after every 1,000 iterations, ensuring a fluid and responsive rendering process.

Insights

  • With the given code you can adjust the size of the triangle to whatever you'd like!
